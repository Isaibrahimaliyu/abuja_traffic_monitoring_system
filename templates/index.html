Abuja Traffic Monitoring System

Flask-based Traffic Data Collection & Analysis

Project Overview

This system collects and analyzes traffic data for major routes within Abuja using OpenStreetMap (OSRM) routing services. Traffic conditions are simulated based on time and day.
